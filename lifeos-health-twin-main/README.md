# LifeOS Health Twin

An intelligent AI-powered health and wellness platform that helps you predict your future health outcomes and make informed lifestyle decisions.

## 📋 About

**LifeOS Health Twin** is a sophisticated platform that uses artificial intelligence to simulate your future health decisions and predict their outcomes. Whether you're considering lifestyle changes, evaluating wellness choices, or planning your health trajectory, LifeOS Health Twin provides data-driven insights and visualizations to help guide your decisions.

### Key Features

- **AI Health Simulation** - Run intelligent simulations on your lifestyle choices and see predicted health outcomes before committing
- **Health Analytics** - Real-time monitoring with AI-driven health scoring and predictive trend analysis
- **Future Health Visualization** - See your possible health futures visualized across fitness, longevity, and wellness metrics
- **Mind Mirror** - Emotion detection and mental stability tracking with actionable insights for mental wellness
- **Food Intelligence** - Snap a photo of your meal and get instant nutritional breakdown with dietary recommendations
- **Time Machine** - See how today's health choices shape your wellness 5, 10, 20 years from now

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher) - [Install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
- npm or bun package manager

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/yourusername/lifeos-health-twin.git
cd lifeos-health-twin
```

2. **Install dependencies:**
```bash
npm install
# or if using bun
bun install
```

3. **Start the development server:**
```bash
npm run dev
# or
bun run dev
```

The application will be available at `http://localhost:5173`

## 🏗️ Tech Stack

- **Frontend Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Form Management**: React Hook Form
- **State Management**: TanStack React Query
- **Routing**: React Router
- **Animations**: Framer Motion
- **Testing**: Vitest
